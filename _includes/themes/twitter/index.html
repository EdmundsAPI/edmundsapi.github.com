How does it work?

 

documentation

Organized around resources, code samples with syntax highlights, easy to explore.

 

inspector

Users can make API calls through edmunds online support, we show them how they differ from documentation.

 

community

Opensource your API documentation on GitHub. The right channel between developers on both side of the API.

 

always tested!

The documentation is always watching your real API and tells you whenever it gets out of sync.
