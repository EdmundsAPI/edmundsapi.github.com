Everything you need to know about an automotive vehicle. The datasets supplied through this API include: 1) Vehicle Specs: trim, engine, transmission, options, torque, MPG, seating capacity, fuel type, colors, configuration ...etc. 2) Vehicle Pricing: TMV®, TCO®, Incentives, and Rebates. 3) Vehicle Service: Maintenance, Recalls, Bulletins, ...etc. 4) Ratings and Reviews: Consumer reviews, Edmunds letter-grade ratings. 5) VIN Decoding. 6) Vehicle Media: Photos and Videos

How does it work?

 

documentation

Organized around resources, code samples with syntax highlights, easy to explore.

 

inspector

Users can make API calls through edmunds online support, we show them how they differ from documentation.

 

community

Opensource your API documentation on GitHub. The right channel between developers on both side of the API.

 

always tested!

The documentation is always watching your real API and tells you whenever it gets out of sync.
