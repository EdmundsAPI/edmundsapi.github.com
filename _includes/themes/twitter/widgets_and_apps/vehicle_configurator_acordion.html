How to embed widget

To embed Vehicle Widget on your page just follow the instructions from Get Code popup.

Example scenario:

  1. Add the following JavaScript files to your page. It's preferably to add them at the bottom of your page for best page performance:

    <script src="http://widgets.edmunds.com/js/edm/sdk.js"></script>
<script src="http://widgets.edmunds.com/js/carconfig/carconfig.js"></script>

  2. Create new instance of the Widget:

    var widget = new EDM.NVC(vehicleApikey, dealerApiKey, {root: 'nvcwidget', baseClass: 'nvcwidget'});

  3. Widget initialisation:

    widget.init({"includedMakes":"all","showVehicles":"ALL","zip":"12345"});

  4. Widget rendering:

    widget.render();

  5. Add the CSS files to your page. It's preferably to add them at the head of your page:

    <link rel="stylesheet" href="http://widgets.edmunds.com/css/carconfig/simple-light.css">
<style>
.nvcwidget{-webkit-border-radius:5px;-moz-border-radius:5px;border-radius:5px;border-width:1px;width:248px;}.nvcwidget .nvcwidget-header{-webkit-border-top-right-radius:5px;-moz-border-radius-topright:5px;border-top-right-radius:5px;-webkit-border-top-left-radius:5px;-moz-border-radius-topleft:5px;border-top-left-radius:5px;}
</style>
Widget custom events

Vehicle widget provides the next methods for working with custom events:


on() - Binds a callback function to an object. The callback will be invoked whenever the event is fired.

// External usage example:
widget.componentName.on('complete', function(options) {
    // this code is executed when the change event is fired by the widget
});

// Internal usage example:
this.componentName.on('complete, function(options) {
    // this code is executed when the change event is fired by the widget
});

off() - Removes a previously-bound callback function from an object. If no event name is specified, all callbacks will be removed.

// External usage example:
widget.componentName.off('complete');

// Internal usage example:
this.componentName.off('complete');

trigger() - Trigger callbacks for the given event. Subsequent arguments to trigger will be passed along to the event callbacks.

this.componentName.trigger('complete', options);

componentName - this is one component of Vehicle Widget.
